# Dinosaur Game

### 简要介绍：
这是一个简易版的Chrome恐龙游戏。

> 展示页面 https://ockhamrazor.github.io/chrome-dinosaur-game/

### 版本说明
> v0.0.1 

> v0.0.2   
1. 修复了除零引起的碰撞机制失效
2. 修复了不能显示满分的bug

### 规则介绍：

#### 基本规则
> 1. 跳跃躲过障碍物

> 2. 每当分数增加300分可能会出现翼龙

> 3. 满分为2000分

#### 键位介绍
> `p` 暂停游戏

> `r` 强制重新开始游戏

> `space` 跳跃 （GameOver时可以触发重新开始）

### 使用说明：

可以通过直接双击打开`index.html`运行
